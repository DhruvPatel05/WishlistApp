# ✨ WishlistApp

WishlistApp is a clean and modern SwiftUI application built using **SwiftData** that allows users to create, manage, and organize their personal wishes seamlessly.

The app demonstrates CRUD operations (Create, Read, Update, Delete) with local persistent storage while maintaining a smooth and elegant iOS user experience. :contentReference[oaicite:0]{index=0}

---

## 🚀 Features

- ➕ Add new wishes instantly
- 📝 Store wishes locally using SwiftData
- 🗑️ Swipe to delete wishes
- 💾 Persistent local storage
- ❤️ Beautiful empty state screen
- 📱 Clean and minimal SwiftUI interface
- ⚡ Fast and responsive user experience
- 🎨 Modern iOS design patterns

## 🚀 Installation

Clone the repository:

```bash
git clone git@github.com:DhruvPatel05/WishlistApp.git
```

Open the project in Xcode:

```bash
open WishlistApp.xcodeproj
```

Run the project using:
- iOS Simulator
- or a physical iPhone device

---

## 📱 Usage

1. Launch the app
2. Tap the ➕ button
3. Add your new wish
4. Save it locally
5. Swipe left to delete wishes
